                           README
Архив форума ***.*****. Загружены все темы, изображения
(png, jpg, jpeg) и файлы, кроме аватаров пользователей. Первая
загрузка произведена в марте 2025 года с помощью Python-скриптов.
Ссылки на файлы локализованы. Архив работает как статический
сайт без веб-сервера. Поиск возможен только по тексту и
названиям файлов через утилиты типа "find".
Темы сохранены в HTML и JSON. В HTML отображаются никнейм,
пост и дата. JSON содержит дополнительные данные (дата
изменения, теги, список пользователей).
Структура упрощена. Темы "About ..." из "Community software"
перенесены в основной раздел.

                         Important
Гиперссылки, оставленные в тексте пользователей форума,
вы открываете на свой страх и риск, как и на самом ***.*****
Данные каждой темы сохранены также в json-файлах. Из них, 
при желании, можно полностью восстановить её изначальный вид.
Директория ./CensorshipCircumventionMethodsSoftware/господа-знатоки.. 
не доступна в github-pages, причина: "Filename too long".
