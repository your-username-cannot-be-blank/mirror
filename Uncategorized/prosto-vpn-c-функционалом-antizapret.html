Не знаю, как работает prostovpn, но я для policy based routing использую следующие костыли:

  • В windows ставлю метрики TAP интерфейсу выше, чем основному (например 3000 и 2000 соответственно):
    Get-NetIPInterface | Format-Table -AutoSize
    Set-NetIPInterface -InterfaceIndex “IfIndex value” -InterfaceMetric “InterfaceMetric value”
  • Запускаю менеджер тунеллей GOST с настроенным SOCKS5 прокси и interface: “OpenVPN TAP”
  • Настраиваю в браузере раширение SwitchyOmega для роутинга доменов на прокси
  • Для программ без возможности ипользовать прокси ипользую Proxifier
  • Для UDP использую в GOST port forwarding
