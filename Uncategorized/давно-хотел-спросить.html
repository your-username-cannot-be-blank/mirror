Служба гудбая стоит в ручную.
Создаём батник файл .bat пишем в него это

Start “” “C:\Program Files\Mozilla Firefox\firefox.exe”
net start GoodbyeDPI
@echo off
set “p1=firefox.exe”
set “p2=goodbyedpi.exe”

cmd /c for /l %%# in () do @tasklist /fi “imagename eq %p1%”^|findstr /v :^|^| (taskkill /f /im “%p2%”^& exit/b)^&^>nul timeout /t 1 /nobreak

У меня это на примере мозилы. Ложем куда нить этот батник и отправляем ярлык на рабочий стол и он будет запускать мозилу и включать службу гудбая, при закрытии браузера служба будет останавливаться. Я себе сделал добавив файл vbs чтоб батник запускался невидимый, в свойствах ярлыка батника можно поставить любую иконку и будут два ярлыка браузера, один обычный, второй с гудбаем, автоматика ) Идея моя, код где то надыбал на просторах. OpenVPN чтоб не держать всегда включённым, я себе сделал 2 батника, один включает, второй выключает, не просто выключает, а останавливает службы, зачем они включены всегда, включённый этот второй адаптер )) Кому надо, вот вам код.
Первый включает

sc config OpenVPNServiceInteractive start= auto
sc config OpenVPNService start= auto
net start OpenVPNServiceInteractive
net start OpenVPNService
start C:"Program Files"\OpenVPN\bin\openvpn-gui.exe

Второй выключает

taskkill /im openvpn-gui.exe
net stop OpenVPNServiceInteractive
net stop OpenVPNService
sc config OpenVPNServiceInteractive start= disabled
sc config OpenVPNService start= disabled

Учитесь пользоваться батниками, нормальная тема )
