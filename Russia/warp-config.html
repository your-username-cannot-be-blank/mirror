Ищешь и качаешь wgcf на гитхабе. В список гудбая добавляешь cloudflareclient.com чтобы прошел регистрационный трафик. Регаешь аккаунт с включенным гудбаем (wgcf register) & (wgcf generate). Должен появиться conf файл. Открываешь, вписываешь параметры для обфускации амнезией, при необходимости (в разделе [interface]):

Jc = 120
Jmin = 25
Jmax = 999
H1 = 1
H2 = 2
H3 = 3
H4 = 4

Меняешь endpoint с доменного имени на айпишник. Можно поискать рабочий через check_cf_warp_endpoint.py (нужен питон). Он создает список айпи, рассортированный по пингу. Пробуешь по очереди пока не найдешь рабочий.
