Here’s a network censorship presence list of Virtual Private Servers and Dedicated Servers providers in Russia.

  • kinokrad.win, meduza.io, windscribe.com are blocked officially with the block record in Unified Register of Domain Names… Containing Information Prohibited for Distribution in the Russian Federation (full list). These kind of blocks are performed on many commercial DPI systems with official blocklist support.
  • news.google.com, play.google.com, adguard.com are unlisted blocks performed only on a government TSPU DPI systems.

— the website is reachable over HTTPS
— the website is not reachable over HTTPS
— the website is reachable only sometimes, unreliable connection

Only all 6 mean that the network is free of censorship. You should prefer these kind of servers if you’re looking for a VPN in Russia.
In case of a censorship test box, one should prefer all 6 .

Unreliable results marked with are usually due to multiple uplink ISP connected to the provider, some of which implement block and the others do not. The same applies to the results where some of TSPU-blocked websites are reachable and the others are not: only part of the traffic is routed via the network with censorship.
