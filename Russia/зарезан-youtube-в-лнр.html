  1. Зайдите на Youtube
  2. Нажмите F12 в браузере, перейдите во вкладку Network (сеть)
  3. Откройте любой видеоролик
  4. Во вкладке Network будет множество запросов на домены вида rr3---sn-5goeen7r.googlevideo.com. Нажмите на любом из них правой кнопкой → Copy URL
  5. Загрузите этот URL с помощью cURL (cURL встроен в Windows 10 и 11, есть на сайте для более старых версий Windows):
    curl -o /dev/null 'https://rr3---sn-5goeen7r.googlevideo.com/videoplayback?expire=…'

$ nslookup rr3---sn-5goeen7r.googlevideo.com
