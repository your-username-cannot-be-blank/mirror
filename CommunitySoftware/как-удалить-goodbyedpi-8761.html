MarkVobl

Файл с exe это не установщик, а сама программа. Запускать ее надо через терминал с параметрами.

Для удобства есть файлы с расширением cmd где уже настроены наиболее типичные параметры запуска программы.
Рекомендую вам запускать через файл 1_russia_blacklist_dnsredir.cmd. Если вы хотите добавить программу в автозагрузку, то можете запустить service_install_russia_blacklist_dnsredir.cmd. Это создаст сервис, который будет работать в фоне и запускаться при включении компьютера.

Файлы cmd вы можете открыть в текстовом редакторе и посмотреть, что они делают.
Для удаления(остановки работающего экземпляра и его удаления из планировщика заданий) есть скрипт service_remove.cmd. После его запуска от имени администратора можно просто удалить папку, если она вам больше не нужна.

Рекламы быть не должно. (Если вы в России). Это программа не VPN и не Proxy.
