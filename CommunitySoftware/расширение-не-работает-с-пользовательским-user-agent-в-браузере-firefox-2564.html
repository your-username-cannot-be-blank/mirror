Ну, значит проблема на стороне АнтиЗапрета.

Для полноты картины давайте посмотрим ошибки в логах после установки “Антицензорити”:

  1. Переключить PAC-скрипт на Антицензорити.
  2. Открыть about:debugging#/runtime/this-firefox.
  3. Найти наше расширение, нажать Inspect/Исследовать и запечатлеть красные записи (ошибки).
  4. Также можно посмотреть в Browser console: Ctrl+Shift+J. Там может быть много незначительных ошибок, не пугайтесь.
