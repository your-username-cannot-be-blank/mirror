Мини утилита для запуска ByeDPI + ProxiFyre под Windows

Требования

  1. Windows 7+, .NET Framework 4.7.2+
  2. ProxiFyre, Windows Packet Filter, Visual C++ Redist 2022
  3. ByeDPI

Использование

  1. Скачать и распаковать ByeDPI и ProxiFyre.
  2. Установить Windows Packet Filter
    Defender может кинуть на него детект, по тем же причинам, что и на WinDivert
  3. Указать пути к exe ByeDPI и ProxiFyre
  4. Настроить параметры запуска ByeDPI
  5. Добавить приложения, которые должны работать через прокси
  6. Вы восхитительны

P.S.: В репозитории и в утилите нет исполняемых файлов Windows Packet Filter, ProxiFyre и ByeDPI, их необходимо скачивать самостоятельно. Писал больше для себя и целью было сделать решение легковесным. Но для тех, кто не хочет этим заниматься, в релизах прикреплен архив All in One под Windows 10+ x64, а-ля сборка
