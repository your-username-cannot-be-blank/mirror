Временная «грязная» версия, определяющая протокол OpenVPN. Выполняет фрагментацию и отправку поддельного пакета для этого VPN-протокола.
Позволяет подключиться к заблокированным VPN-сервисам по протоколу OpenVPN TCP, таким как ProtonVPN.

Для скачивания файла необходима учётная запись Github.

Включение определения задаётся параметром --openvpn. По умолчанию определение выполняется только на порту 443, добавить дополнительные порты можно опцией --port 1234, где 1234 — номер порта.

Рекомендуемый вариант запуска — дописать опцию --openvpn в скрипт 1_russia_blacklist.cmd.
