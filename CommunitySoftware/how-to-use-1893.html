Здесь находится информация о работе с DPITunnel для Android и для Linux

github.com

GitHub - zhenyolka/DPITunnel-android: https://github.com/nomoresat/DPITunnel-android

https://github.com/nomoresat/DPITunnel-android

Минимальная версия: Android 5.0
Для работы необходим ROOT из-за использования raw socket.
Поддерживаются профили для разных Wi-Fi сетей и один для мобильного инета. Так же есть профиль “по-умолчанию”, если под подключение нет отдельного профиля.
В программу зашит профиль Auto TTL universal, работающий для большинства операторов, в том числе для внереестровых блокировок и замедлений.
Есть функция автонастройки: кнопка “подобрать настройки” в параметрах профиля.

github.com

GitHub - zhenyolka/DPITunnel-cli: https://github.com/nomoresat/DPITunnel-cli

https://github.com/nomoresat/DPITunnel-cli

Версия для Linux предназначена для ПК и роутеров на Linux. Может работать в режиме HTTP и transparent прокси. Transparent режим позволяет установить программу на роутер и обходить блокировки для всей локальной сети без необходимости индивидуальной настройки клиентов. Подробнее в README.

Here is information about working with DPITunnel for Android and for Linux

github.com

GitHub - zhenyolka/DPITunnel-android: https://github.com/nomoresat/DPITunnel-android

https://github.com/nomoresat/DPITunnel-android

Minimum version: Android 5.0
It requires ROOT to work due to the use of raw socket.
Profiles for different Wi-Fi networks and one for mobile Internet are supported. There is also a “default” profile, if there is no separate profile for the connection.
The program has an Auto TTL universal profile that works for most operators, including blocks and slowdowns.
There is an auto-tuning function: the “set automatically” button in the profile parameters.

github.com

GitHub - zhenyolka/DPITunnel-cli: https://github.com/nomoresat/DPITunnel-cli

https://github.com/nomoresat/DPITunnel-cli

The Linux version is intended for PCs and Linux routers. Can work in HTTP and transparent proxy mode. Transparent mode allows you to install the program on the router and bypass blocking for the entire local network without the need for individual client settings. Read more in README.
