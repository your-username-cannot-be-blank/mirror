На чистую систему перенесите скрипт antizapret-vpn.sh в root. Я это сделал через MobaXterm.

Можно сделать еще так, но тогда вместе со скриптом будет папка с документацией.

git clone https://github.com/GubernievS/AntiZapret-VPN-Container.git

cd AntiZapret-VPN-Container

chmod +x ./antizapret-vpn.sh && ./antizapret-vpn.sh
