kamlipoltu

похожая проблема видимо имеет старые корни и обсуждалась тут. Найденное решение проблемы:

  1. Find your Network Manager vpn configuration file (mine is in /etc/NetworkManager/system-connections; if you have a lot of them and filenames do not help much in finding the right one, use grep -i “id=yourmnemonicname” *)
  2. Under the [vpn] section add the line:
    tls-cipher=DEFAULT:@SECLEVEL=0
  3. Reload the configuration with the command: nmcli connection reload
