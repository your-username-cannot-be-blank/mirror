Если роутер позволяет указать другой DNS с привязкой к домену, то это выход.

Т.е. “для 05cdn.wf использовать адрес DNS провайдера”

Такое умеют прошивки Keenetic, а также любые прошивки, где у пользователя есть доступ к настройкам dnsmasq (OpenWrt, Padavan).

Либо поднять где-то в локалке (на роутере или на NAS) Adguard Home и настроить всё так, чтобы роутер резолвил через AGH, а уже AGH использовал DNS провайдера для конкретнорго домена и DNS АнтиЗапрета для всего остального.

Либо в Windows поставить YogaDNS и в нём создать правило, чтобы по умолчанию использовался DNS АнтиЗапрета, а для конкретного домена - провайдерский.
