А Tampermonkey есть в большинствах мобильных браузерах.

Обход блокировок Рунета не работает в мобильном Хроме, т.к. там нет поддержки расширений совсем. Не работает оно и на FireFox под Андроид, т.к. там нет унжного API (см. Дополнение для Firefox на Android · Issue #86 · anticensority/runet-censorship-bypass · GitHub).

Если даже расширение поддерживает какую-либо платформу, то конфликта с Tampermonkey на ней не должно возникать.
