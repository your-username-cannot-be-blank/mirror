If you live in a country with internet censorship and have IPv6 connectivity from ISP, you can use special DNS server (DNS64) which will redirect all IPv4 addresses over IPv4→IPv6 proxy called NAT64.
This allows to access IPv4-only resources via free IPv6 proxy.

I found two public DNS64+NAT64 gateways:

TREX Regional Exchanges Oy

Public Service: DNS64 Name Servers

Public NAT64 service description at TREX.

go6lab

NAT64/DNS64 public test - go6lab

Go6lab is hosting a variety of NAT64/DNS64 solutions, open for general Internet public for testing. Disclaimer: This setup is not intended for performance testing, just to see how NAT64/DNS64 operates, what applications breaks and what are...

The configuration is trivial: just set up their DNS server as the primary server in the OS network configuration.
