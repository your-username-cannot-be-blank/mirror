Вообще, построение mesh сети для массового обхода блокировок лишено смысла.

  1. Для любых mesh-сетей работающих на базе интернета характерной чертой будут массовые соединения между резидентскими ip.

  2. Для любых mesh-сетей на базе телефонии (welcome to 20 years ago) есть оператор для которого массовые звонки на одни и те же номера яно будут также заметны. Да и я сомневаюсь, что IP трафик можно убедительно замаскировать под телефон.

  3. Для любых mesh-сетей на базе альтернативных беспроводных технологий (т.е. радио, оптику я в данном случае всерьёз воспринимать не могу) есть ГКРЧ, которая по борьбе с такими любителями имеет опыт 100 лет уже.
